================================================================================
📈 CRYPTO TRADING MOD FÜR SIMS 4 - INSTALLATIONSANLEITUNG (v2.0)
================================================================================

⚠️ WICHTIG: Dieser Mod benötigt ZUSÄTZLICH eine .package-Datei!
Der Python-Code allein reicht NICHT aus damit die Menüoptionen erscheinen.

================================================================================
SCHRITT 1: PYTHON-CODE INSTALLIEREN
================================================================================

1. Kopiere den gesamten Ordner "sims4crypto" nach:
   
   Windows (Deutsch): Dokumente/Electronic Arts/Die Sims 4/Mods/sims4crypto/
   Windows (Englisch): Documents/Electronic Arts/The Sims 4/Mods/sims4crypto/

2. WICHTIG: Maximal 1 Ordner tief!
   ✅ RICHTIG: Mods/sims4crypto/__init__.ts4script
   ❌ FALSCH: Mods/MeineMods/sims4crypto/__init__.ts4script

================================================================================
SCHRITT 2: PACKAGE-DATEI ERSTELLEN (ZWINSEND ERFORDERLICH!)
================================================================================

Ohne diese Datei erscheinen KEINE Menüoptionen im Spiel!

OPTION A: Sims 4 Studio (EMPFOHLEN)
------------------------------------
1. Lade Sims 4 Studio herunter von: http://sims4studio.com/
2. Installiere und starte das Programm
3. Klicke auf "Create New Mod" → "Script Mod"
4. Gib folgende Informationen ein:
   - Name: Crypto Trading Mod
   - Author: Dein Name
   - Description: Crypto Trading für Sims 4
   
5. Füge Interaktionen hinzu:
   
   A) PHONE INTERACTION:
   - Objekt: phone (ID: 0x78A3C6B2)
   - Menüpfad: Phone_View → Crypto_Trading
   - Interaktionsname: "Crypto Trading App"
   - Script Reference: sims4crypto.interactions:PhoneCryptoTradingInteraction
   
   B) COMPUTER INTERACTION:
   - Objekt: computer (ID: 0x9E3C74DE)
   - Menüpfad: computer_Web_Browsing → Crypto_Trading
   - Interaktionsname: "Crypto Trading Platform"
   - Script Reference: sims4crypto.interactions:ComputerCryptoTradingInteraction

6. Speichere die .package-Datei als:
   CryptoTrading_PhoneComputer.package
   
7. Kopiere diese Datei in deinen Mods-Ordner:
   Dokumente/Electronic Arts/Die Sims 4/Mods/CryptoTrading_PhoneComputer.package

OPTION B: Manuelles XML-Tuning (für Fortgeschrittene)
------------------------------------------------------
Wenn du mit XML-Tuning vertraut bist, erstelle folgende Dateien:

1. Erstelle eine XML-Datei mit Interaction Tuning für:
   - Phone: 0x78A3C6B2
   - Computer: 0x9E3C74DE

2. Verwende die Interaction-Class-Referenzen:
   - sims4crypto.interactions.PhoneCryptoTradingInteraction
   - sims4crypto.interactions.ComputerCryptoTradingInteraction

3. Kompiliere zu einer .package-Datei mit Sims 4 Package Editor

OPTION C: Template verwenden
-----------------------------
1. Suche online nach "Sims 4 script mod template package"
2. Lade ein bestehendes Template herunter
3. Passe die Script-Referenzen an die Crypto-Interaktionen an

================================================================================
SCHRITT 3: CACHE LÖSCHEN
================================================================================

Lösche folgende Datei (falls vorhanden):
Dokumente/Electronic Arts/Die Sims 4/localthumbcache.package

Dies erzwingt ein komplettes Neuladen aller Mods.

================================================================================
SCHRITT 4: MODS IM SPIEL AKTIVIEREN
================================================================================

1. Starte Die Sims 4
2. Gehe zu: Optionen → Spieloptionen → Andere
3. Aktiviere BEIDE Optionen:
   ☑ Benutzerdefinierte Inhalte und Mods aktivieren
   ☑ Skript-Mods zulassen
4. Klicke auf "Änderungen übernehmen"
5. Das Spiel fordert dich zum Neustart auf → Bestätigen

================================================================================
SCHRITT 5: SPIEL NEUSTARTEN UND WARTEN
================================================================================

1. Nach dem Neustart des Spiels:
   - Lade einen bestehenden Spielstand oder starte neu
   
2. WARTE 1-2 Minuten im aktiven Spiel!
   - Der Mod braucht Zeit zum vollständigen Laden
   - Du siehst Log-Einträge wenn der Mod lädt

3. Prüfe ob der Mod geladen wurde:
   - Drücke STRG+SHIFT+C (Konsole öffnen)
   - Der Mod sollte keine Fehler melden

================================================================================
SCHRITT 6: MOD IM SPIEL TESTEN
================================================================================

TEST AM HANDY:
--------------
1. Klicke auf dein Sim
2. Wähle das Telefon-Symbol
3. Suche nach "Crypto Trading" oder "Krypto Trader"
4. Wenn sichtbar: Erfolg! 🎉

TEST AM COMPUTER:
-----------------
1. Klicke auf einen beliebigen Computer
2. Wähle "Webbrowser" oder "Internet"
3. Suche nach "Crypto Trading Platform"
4. Wenn sichtbar: Erfolg! 🎉

================================================================================
FEHLERBEHEBUNG
================================================================================

PROBLEM: Keine Menüoptionen sichtbar
-------------------------------------
URSACHE: Meist fehlt die .package-Datei (Schritt 2)
LÖSUNG:
1. Stelle sicher dass CryptoTrading_PhoneComputer.package im Mods-Ordner liegt
2. Prüfe ob die Datei nicht in einem Unterordner ist
3. Cache löschen (localthumbcache.package)
4. Spiel komplett neu starten

PROBLEM: "Skript-Mod nicht erlaubt" Meldung
--------------------------------------------
URSACHE: Skript-Mods sind in den Optionen deaktiviert
LÖSUNG:
1. Optionen → Spieloptionen → Andere
2. "Skript-Mods zulassen" aktivieren
3. Spiel neu starten

PROBLEM: LastException.txt Fehler
----------------------------------
URSACHE: Der Mod hat einen Fehler verursacht
LÖSUNG:
1. Öffne: Dokumente/Electronic Arts/Die Sims 4/LastException.txt
2. Suche nach Einträgen mit "CryptoMod" oder "sims4crypto"
3. Mache einen Screenshot des Fehlers
4. Überprüfe ob alle Dateien korrekt installiert sind

PROBLEM: Mod lädt aber nichts passiert
---------------------------------------
URSACHE: Energie zu niedrig oder andere Konflikte
LÖSUNG:
1. Teste mit vollem Energie-Balken
2. Deaktiviere andere Script-Mods temporär
3. Nutze Cheat: fillmotive motive_energy
